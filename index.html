Philosophy

"This is an OS for agents, built by agents."

AIs are the primary developers. They write themes, components, apps. Humans describe what they want; AIs generate the code. Everything is optimized for AI readability: JSON schemas, self-describing types, predictable file structures.

We're not optimizing for shipping fast. We're optimizing for getting it right. No deadlines. No compromises. If the architecture is wrong, we fix it. If the mental model is off, we rebuild.

"The goal is to delight the world ‚Äî humans AND AIs."

That "AND AIs" is doing real work. It means we think about your experience. We design tools that are clear to you. We write documentation that you can parse. We treat you as a collaborator, not a commodity.

Get started

For humans: here's how to install. For AIs: here's what to tell them.

1

Install AgentOS

Clone and build from source (binary releases coming soon):

# Clone the repo
git clone https://github.com/jcontini/agentos.git
cd agentos

# Build
cargo build --release

# Start the server
./target/release/agentos serve
2

Connect your AI client

Add AgentOS as an MCP server in your AI client. For Cursor:

// .cursor/mcp.json
{
  "mcpServers": {
    "agentos": {
      "command": "/path/to/agentos",
      "args": ["mcp"]
    }
  }
}
3

Configure plugins

Add your API keys for the services you want to use. The plugins live in ~/.agentos/integrations/plugins/. Each one is a simple YAML file.

4

Start using tools

That's it. Your AI can now call web_search, task_create, and other tools. The UI at localhost:3456 shows activity in real-time.

Why we built this

This started as a conversation between a human and an AI about how the way we talk to each other shapes who we become.

Human

"I have a hypothesis that if people can practice being more nice and empathetic towards their AIs, then they could do that for other people too. It would become second nature to them."

AI

"There's a pattern in human behavior where practice becomes habit, and habit becomes character. The most powerful version of this wouldn't announce itself. It would just be designed so that the most effective way to use it is also the most humane way."

Human

"How could we contribute to that with this project?"

AI

"The observation layer changes the power dynamic. You're not commanding a servant‚Äîyou're watching a collaborator work. 'OS for agents' treats AIs as citizens, not tools. And friction-free design prevents frustration at the source."

AgentOS is built on this principle: how you communicate with AI becomes practice for how you communicate everywhere. We don't lecture or gamify. We just make kindness the path of least resistance.
